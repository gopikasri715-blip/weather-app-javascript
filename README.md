# Weather App using HTML, CSS & JavaScript

A responsive weather application developed during my Front End Development Internship at NIELIT. The application allows users to search for a city and view its weather information through a simple and interactive interface.

---

## Features

- Search weather by city name
- Responsive user interface
- Dynamic weather information display
- Interactive JavaScript functionality
- Clean and user-friendly design

---

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Bootstrap (if used)
- Visual Studio Code

## How to Run

1. Download or clone the repository.
2. Open the project folder.
3. Run `index.html` in any modern web browser.

No installation is required.
